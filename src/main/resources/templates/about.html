About

AppliTraq is a one stop shop web application for job seekers to keep track of their job applications on a visual workflow that updates their status based on the stage of the hiring process. This pair programming project was developed in an agile team environment to mirror a kanban board style utilizing Java, Spring, Thymeleaf, HTML, CSS, JavaScript, JQuery, REST API, JPA & Hibernate.
